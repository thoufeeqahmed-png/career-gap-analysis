# Career Progression and Promotion Gap Analysis for Retention Optimization at Palo Alto Networks

## Project Overview

This project analyzes employee career progression patterns to identify promotion gaps, role stagnation, and retention opportunities. The objective is to help organizations proactively address career development issues before employees become disengaged or leave the company.

## Features

* Career Path Clustering using K-Means
* Promotion Gap Analysis
* Retention Opportunity Identification
* Manager Stability Insights
* Interactive Streamlit Dashboard

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Matplotlib
* Seaborn

## How to Run

1. Install required packages:
   pip install -r requirements.txt

2. Run the Streamlit app:
   streamlit run app.py

## Dataset

Employee career progression and HR analytics dataset containing promotion history, role tenure, training participation, managerial continuity, and retention indicators.
